# Drupal Assistant

Generate the credentials that connect the Cinatra Drupal module (cinatra) to your workspace, so editors can launch the Cinatra assistant from any Drupal node page. This surface mints the API key the module sends as a bearer token and shows the Cinatra URL to paste into the module's configuration.

## Capabilities

- Generate and rotate the API key used by the Cinatra Drupal widget module
- Copy the Cinatra URL and key to paste into the module's configuration form
- Regenerate credentials, which immediately invalidates the previous values
- Surface setup instructions for installing and wiring the widget module
